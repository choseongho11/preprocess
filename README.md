# 회의록 요약 전처리기 (Legislative Meeting Minutes Parser & Summarizer)

본 프로그램은 지방의회 등의 회의록 PDF 파일을 업로드하면, 의원들의 발언만 추출하여 설정된 LLM(대규모 언어 모델)을 통해 자동으로 요약해주는 웹 애플리케이션입니다.

## 다른 컴퓨터에서 실행하는 방법

### 1. 프로그램 파일 복사
현재 작업 중인 `회의록 요약 전처리` 폴더 전체를 압축(ZIP)하거나 USB, 클라우드(구글 드라이브 등)를 통해 다른 컴퓨터로 복사합니다.

### 2. Python 설치
- 다른 컴퓨터에 Python이 설치되어 있지 않다면, [Python 공식 홈페이지](https://www.python.org/downloads/)에서 Python 3.8 이상의 버전을 다운로드하여 설치합니다.
- **설치 시 주의사항:** 설치 화면 하단의 **"Add Python to PATH"** (또는 "Add python.exe to PATH") 옵션을 반드시 체크한 후 설치를 진행하세요.

### 3. 필수 패키지 설치
1. 복사해 온 `회의록 요약 전처리` 폴더를 엽니다.
2. 폴더 주소창(경로창)에 `cmd`를 입력하고 엔터를 치거나, 터미널을 열고 해당 폴더 경로로 이동(`cd 경로`)합니다.
3. 아래 명령어를 입력하여 필요한 라이브러리들을 설치합니다:
   ```bash
   pip install -r requirements.txt
   ```

### 4. 환경 변수 설정 (.env 파일)
프로그램이 정상적으로 동작하기 위해서는 `.env` 파일에 LLM 연결 정보가 있어야 합니다. 
만약 다른 컴퓨터로 복사할 때 숨김 파일인 `.env`가 복사되지 않았다면, 폴더 안에 새 파일 `.env`를 만들고 아래와 같이 내용을 채워주세요. (기존 컴퓨터의 내용과 동일하게 맞춰주시면 됩니다.)

```env
# 로컬 LLM 설정
LLM_BASE_URL=http://115.137.55.154:90/v1
LLM_API_KEY=발급받은_API_KEY를_입력하세요
LLM_MODEL=gemm4 e4b

# Flask 설정
FLASK_SECRET_KEY=change-this-secret-key
```

### 5. 프로그램 실행
패키지 설치가 완료되면, 같은 터미널 창에서 아래 명령어를 입력하여 서버를 실행합니다:
```bash
python app.py
```
*(Mac 환경에서는 `python3 app.py`를 입력해야 할 수도 있습니다.)*

### 6. 접속하기
터미널에 실행 완료 메시지가 나타나면, 웹 브라우저(크롬, 엣지 등)를 열고 주소창에 아래 주소를 입력하여 접속합니다:
- **http://localhost:5001** 
- 또는 **http://127.0.0.1:5001**

이제 다른 컴퓨터에서도 동일하게 PDF 파일을 업로드하고 요약 기능을 사용할 수 있습니다!
